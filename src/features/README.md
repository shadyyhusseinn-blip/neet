# Features Structure

هذا المجلد يحتوي على الميزات المنظمة حسب الوظيفة (Feature-based Structure).

## الهيكل المقترح

```
features/
├── auth/              # ميزات المصادقة
│   ├── components/    # مكونات المصادقة
│   ├── hooks/         # hooks مخصصة
│   ├── services/      # خدمات المصادقة
│   └── pages/         # صفحات المصادقة
├── admin/             # ميزات الإدارة
│   ├── dashboard/     # لوحة التحكم
│   ├── bookings/      # إدارة الحجوزات
│   ├── clients/       # إدارة العملاء
│   ├── galleries/     # إدارة المعارض
│   └── settings/      # الإعدادات
├── public/            # ميزات الصفحات العامة
│   ├── landing/       # الصفحة الرئيسية
│   ├── portfolio/     # معرض الأعمال
│   ├── packages/      # الباقات والأسعار
│   └── contact/       # صفحة التواصل
└── staff/             # ميزات الموظفين
    ├── dashboard/     # لوحة تحكم الموظفين
    ├── tasks/         # إدارة المهام
    └── schedule/      # الجدول الزمني
```

## كيفية الاستخدام

عند إضافة ميزة جديدة:
1. أنشئ مجلد جديد داخل features/
2. قم بتنظيم الملفات حسب الهيكل أعلاه
3. استخدم المكونات المشتركة من `src/shared/ui/`
4. استخدم الـ hooks والخدمات من `src/core/`

## ملاحظة

هذا الهيكل جاهز للاستخدام في المستقبل. حالياً، الملفات لا تزال في `src/pages/` ويمكن نقلها تدريجياً إلى هذا الهيكل الجديد.
